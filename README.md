# vue-cnode-mobile
学习vue，搭建cnode社区，以前有搭过[react-cnode-mobile](https://github.com/soulcm/react-cnode-mobile)，但只有一个大的壳子就阉割了，因为很多react坑都已经踩过，就没往下继续了

现在学习vue和移动端新知识，重新搭建cnode


知识点：
* vue2
* vuex2
* vue-router2
* 移动端开发
* fetch
* es6 es7
* less 不用sass的理由你懂的，那个难安装


运行步骤
```
npm install

npm run dev
```

访问 http://127.0.0.1:8091

服务启动后，手机端可访问本机ip查看


吐槽: vscode对vue的支持好差
